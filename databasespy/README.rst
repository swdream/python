What is MySQLdb?
-----------

    MySQLdb is an interface for connecting to a MySQL database server from
    Python. It implements the Python Database API v2.0 and is built on top of
    the MySQL C API.

How to install the MySQLdb?::
-----------

    sudo apt-get install MySQLdb
